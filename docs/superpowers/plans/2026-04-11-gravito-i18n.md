# Gravito i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace hardcoded page-layer messages with `gravito i18n` while keeping API responses English-only.

**Architecture:** Use `@gravito/cosmos` to resolve locale once per request, load per-feature message catalogs, and expose `locale` plus `messages` through shared Inertia props. API controllers stay English-only and do not depend on the i18n runtime.

**Tech Stack:** TypeScript 5.x, Bun, Inertia, `@gravito/cosmos`, bun:test

---
