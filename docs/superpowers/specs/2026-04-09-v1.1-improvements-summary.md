### 1. Dashboard/SdkApi Domain 層設計決定

**決策**：正式決定兩個模組無需 Domain 層

#### 決策依據

**Dashboard 模組**（讀取聚合）
```
特性：純讀操作，無業務邏輯
理由：
  ❌ 無不變式（沒有「允許或禁止什麼」的規則）
  ❌ 無聚合根（不定義或管理業務實體）
  ✅ 符合 CQRS 讀側設計
  ✅ 應用層職責（多個資料源聚合）

分層結構：
  Application Service → 聚合邏輯
  Infrastructure Service → 查詢邏輯
  Presentation Controller → HTTP 映射
```

**SdkApi 模組**（認證代理）
```
特性：驗證金鑰、代理請求，無核心業務
理由：
  ❌ 無業務邏輯（純認證 + 轉發）
  ❌ 無聚合根（無自有實體）
  ✅ 中間件職責（API 閘道）
  ✅ 框架層設計

分層結構：
  Application Service → 驗證邏輯
  Infrastructure Middleware → AppAuthMiddleware
  Presentation Controller → HTTP 端點
```

## 💡 Week 2：事件架構與完整圖表

### 1. Domain Events 實現

**目標**：建立完整的事件驅動架構

#### 新增事件類（Credit 模組）

| 事件 | 用途 | 用戶故事 |
|------|------|--------|
| **CreditDeductedEvent** | 額度被扣除 | 用戶使用 API 時，記錄成本、觸發警告 |
| **CreditToppedUpEvent** | 額度被充值 | 用戶購買額度，記錄、清除警告 |
| **LowBalanceAlertEvent** | 低額度警告 | 額度 < 閾值，發送通知、準備自動充值 |

#### 事件類結構

```typescript
// 基類：DomainEvent（src/Shared/Domain/DomainEvent.ts）
export abstract class DomainEvent {
  readonly eventId: string              // UUID
  readonly aggregateId: string           // Aggregate ID
  readonly eventType: string             // 事件名稱
  readonly occurredAt: Date              // 發生時間
  readonly version: number               // 版本（支援演進）
  readonly data: Record<string, unknown> // 負載

  abstract toJSON(): Record<string, unknown>
}

// 分發器：DomainEventDispatcher（src/Shared/Domain/DomainEventDispatcher.ts）
export class DomainEventDispatcher {
  static getInstance(): DomainEventDispatcher
  on(eventType: string, handler: EventHandler): void
  async dispatch(event: DomainEvent): Promise<void>
  async dispatchAll(events: DomainEvent[]): Promise<void>
}

// 實現：CreditDeductedEvent（src/Modules/Credit/Domain/Events/CreditDeductedEvent.ts）
export class CreditDeductedEvent extends DomainEvent {
  readonly creditAccountId: string
  readonly orgId: string
  readonly amount: string
  readonly remainingBalance: string
  readonly reason: 'api_call' | 'manual_deduction' | 'refund_reversal'
  
  constructor(...)
  toJSON(): Record<string, unknown>
}
```

#### Aggregate 增強

**CreditAccount 修改**：

```typescript
// 新增：Domain Events 收集
private readonly domainEvents: DomainEvent[] = []

// 新增：事件發佈方法
private addDomainEvent(event: DomainEvent): void
getDomainEvents(): DomainEvent[]
clearDomainEvents(): void

// 增強：applyDeduction() 自動生成事件
applyDeduction(amount: string): CreditAccount {
  const newAccount = new CreditAccount({...})
  
  // 自動發佈 CreditDeductedEvent
  newAccount.addDomainEvent(new CreditDeductedEvent(...))
  
  // 若低於閾值，發佈 LowBalanceAlertEvent
  if (newBalance.isLessThanOrEqual(threshold)) {
    newAccount.addDomainEvent(new LowBalanceAlertEvent(...))
  }
  
  return newAccount
}
```
