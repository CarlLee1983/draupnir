# List models (Azure OpenAI)

## HTTP Request

`GET /openai/openai/deployments/{deployment-id}/models`

### Parameters

| Name | In | Type | Description |
| --- | --- | --- | --- |
| deployment-id | path | string | Azure deployment ID |
| api-version | query | string |  |

### Responses

#### 200
Successful response

#### 400
Bad request

#### 500
Internal server error
