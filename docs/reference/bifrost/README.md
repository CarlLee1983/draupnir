# Bifrost OpenAPI 快照

本目錄存放 **Bifrost AI Gateway** 的 OpenAPI 定義副本（與 Draupnir 自身的 `docs/openapi.yaml` 無關）。

| 檔案 | 說明 |
|------|------|
| [`openapi.json`](./openapi.json) | 自官方匯入的完整規格；`scripts/convert_openapi.js` 預設讀取此檔，產出 `../bifrost-api/*.md`。 |

官方最新版：[Bifrost OpenAPI JSON](https://docs.getbifrost.ai/openapi/openapi.json)（亦見 [`../Bifrost.md`](../Bifrost.md)）。
