# Milestone v1.2 Archive: Requirements

**Dashboard 分析和報告 — 為多角色使用者提供完整的 API 使用分析和每月決算報告**

---

## v1.2 Requirements Status

### Data Correctness & Permission Foundation (先決條件 — 第 1 階段)

- [x] **DASHBOARD-P1**: 移除 Admin Dashboard 的硬編碼樣本數據
  - **Outcome**: `AdminDashboardPage.ts` created, real stats from `GetKpiSummaryService` integrated.
  - **Verification**: Admin sees real organization cost data.

- [x] **DASHBOARD-P2**: 修復日誌 DTO 字段名稱不匹配
  - **Outcome**: `UsageLogDTO` Zod schema implemented, unified camelCase mapping across service and React.
  - **Verification**: Token charts show correct input/output token counts.

- [x] **DASHBOARD-P3**: 實施 MEMBER 角色的每用戶金鑰範圍
  - **Outcome**: `DashboardKeyScopeResolver` implemented, enforcing key-level isolation for MEMBER role.
  - **Verification**: Multiple members, permission isolation test passed.

### Usage Overview (使用概覽)

- [x] **DASHBOARD-01**: 使用者可看到組織的 7/30/90 天成本、請求、代幣摘要卡
  - **Outcome**: 4 KPI cards implemented with dynamic time window selection.
  - **Verification**: Metrics update correctly when switching between 7, 30, and 90-day windows.

- [x] **DASHBOARD-02**: 使用者可在時間窗口內查看成本隨時間趨勢折線圖
  - **Outcome**: AreaChart (Recharts) implemented with daily cost aggregation.
  - **Verification**: Cost trend displays correctly over time windows.

### Cost Analysis (成本分析)

- [x] **DASHBOARD-03**: 使用者可看到成本按模型分解的柱狀圖
  - **Outcome**: BarChart (Recharts) showing top 10 models by cost (DESC).
  - **Verification**: Model distribution accurately reflects actual spend.

- [x] **DASHBOARD-04**: 工程師可看到代幣使用（輸入 vs 輸出）堆疊面積圖
  - **Outcome**: Stacked AreaChart implemented for input vs output tokens.
  - **Verification**: Input/output ratio is visible and non-zero.

- [x] **DASHBOARD-05**: 使用者可看到模型比較表（成本、請求數、平均延遲）
  - **Outcome**: Sortable table implemented for detailed model comparisons.
  - **Verification**: Table data matches KPI summary cards.

### Monthly Report (每月決算)

- [x] **DASHBOARD-06**: 每月自動生成高層決算 PDF 報告
  - **Outcome**: PDF generation via `window.print()` implemented. Added % change badges (differentiator).
  - **Verification**: PDF report is clean, nav-hidden, and contains all essential analytics.
